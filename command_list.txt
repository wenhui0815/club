* VScode指令

	- 編譯指令： Ctrl + alt + B
	- 開啟終端機： Ctrl + Shift + ~
	- RUN指令： 輸入./檔名.exe後按下你的Enter鍵後執行
	- 編譯多個檔案： g++ -g 要共同編譯的檔案1 要共同編譯的檔案2 要共同編譯的檔案3 -o 執行黨名稱(自訂).exe

* Git/GitHub基本指令

	- git clone "repository link（你的雲端網址）" : 連結資料夾與GitHub
	- git add "檔案名稱" ： 將檔案傳到git中
	- git commit -m "存檔名稱" ： 存檔！
	- git push ： 把存檔推到雲端
	- git pull ： 把存檔從雲端拉下來

* 命令列（終端機）指令簡介

	- tab :快速輸入鍵，這超重要！！！！

	- cd (路徑) ：前往某個資料夾
	- . ：當前資料夾
	- .. ：上一個資料夾
		ex:
			cd .. ：前往上一個資料夾
			cd C: ：前往C槽
			git add . ：將此資料夾的檔案加入git中
	
	- 上方向鍵 ：之前用過的指令
	- 下方向鍵 ：之前用過的下一個指令
		ex:
			可以利用上下方向鍵找到之前打過的指令
